# Aegis Lessons Learned

## Security Containment
- **Problem:** Credentials leaking from local.env into capability handlers.
- **Fix:** `scripts/execute_mode.sh` must clear provider env vars before invoking capability wrappers. Verified in `test_secret_containment.sh`. Do not alter this isolation.

## JSON Artifact Parsing
- **Problem:** Substrate outputs failing regex extraction when markdown fences contain spaces.
- **Fix:** Use the strict awk-based boundary marker reader established in `raw_llm.sh`. Do not substitute with raw `grep`.
